# ナンプレ ソルバー 🔢

バックトラッキング法（再帰探索）でナンプレ（数独）を自動的に解くソルバーです。
Web版 と Windows デスクトップ版の両方があります。

## 🌐 Web版（GitHub Pages）

**▶ [ライブデモを開く](https://altairu.github.io/nanpure/)**

ブラウザで動作するため、インストール不要です。

### 機能
- ⚡ **即時に解く** — バックトラッキング法で瞬時に解を表示
- 🎬 **アニメーション** — 解く過程をリアルタイムに可視化（バックトラックも表示）
- 🗑 **クリア** — 全マスをリセット
- ❌ **解なし検知** — 矛盾がある問題はエラーメッセージを表示
- 📱 **レスポンシブ対応** — PC・スマホ両対応

## 🖥 デスクトップ版（Windows）

`dist/ナンプレソルバー.exe` をダブルクリックで起動（インストール不要）。

### ビルド方法

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "ナンプレソルバー" nanpure_solver.py
```

## 🔧 アルゴリズム

**バックトラッキング法（再帰探索）**

1. 空マスを左上から順に探す
2. 1〜9 の数字を試す（行・列・3×3 ブロックのルールを検証）
3. 置けた場合は次の空マスへ再帰
4. どの数字も置けない場合は直前のマスに戻る（バックトラック）
5. 全マスが埋まれば完成
